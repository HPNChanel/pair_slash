# Example Output Shape

## Plan Mode

- Objective
- Relevant Memory Constraints
- Systems Affected
- Preconditions
- Step-by-Step Plan
- Validation and Monitoring Plan
- Rollback Plan
- Risks / Blast Radius
- Proposed File Touch List
- MEMORY_CANDIDATE

## Default Mode

- Done
- Files Changed
- Operational Impact
- Validation Run / Validation Needed
- Rollback Notes
- Remaining Risks
- MEMORY_CANDIDATE
